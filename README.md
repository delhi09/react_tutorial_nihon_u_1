# 実行方法

```shell
$ git clone git@github.com:delhi09/react_tutorial_nihon_u_1.git
$ cd react_tutorial_nihon_u_1
$ npm start
```

→ [http://localhost:3000](http://localhost:3000)でアプリが起動します。
